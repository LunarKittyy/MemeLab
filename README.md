# Meme Lab

A layer-based image and meme editor that runs entirely in your browser. No account, no uploads, no nonsense.

## [Use Here](https://lunarkittyy.github.io/MemeLab/)

## Features

- **Layers** - text, images, and shapes with drag-to-reorder
- **Text** - custom fonts, outline, background box, per-character spacing
- **Images** - crop, flip, exposure, background removal (AI)
- **Shapes** - solid color, blur, or pixelate/censor blocks
- **Transform** - move, resize, rotate, opacity on everything
- **Export** - save as PNG at 1x, 2x, or 4x resolution
- **Undo/redo** - full history
- **Autosave** - project saves locally in your browser automatically, nothing ever leaves your device
